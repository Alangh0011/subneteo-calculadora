# Calculadora IP de Subneteo en C#

Esta calculadora IP de subneteo en C# es una herramienta simple para ayudarte a realizar cálculos de subneteo en redes IP.

## Funcionalidades

- **Cálculo de Subredes:** Ingresa una dirección IP y la máscara de subred, y la calculadora te proporcionará información detallada sobre las subredes resultantes.
  
- **Validación de Direcciones IP:** Asegúrate de que las direcciones IP ingresadas sean válidas antes de realizar cálculos.

## Requisitos del Sistema

- [Visual Studio](https://visualstudio.microsoft.com/es/) (o cualquier otro entorno de desarrollo compatible con C#).
